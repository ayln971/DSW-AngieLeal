La Interfaz de usuario e interación humano ordenador

Este sitio web tiene como proposito apoyar la formación de
los estudiantes de la universidad del Estado, por medio de
recursos educativos digitales, con el fin de que se comprenda
el tema de la interfaz del usuario e interación humano ordenador

Interfaz del usuario

es el medio donde las personas se relacionan con las maquinas,
computadoras, equipos moviles, entre otros, un ejemplo de esto seria
la pantalla principal de Windows o las diferentes pantallas de los
programas, tambien puede ser el mause en caso de la parte fiscia
del computador (hardware).

interación humano ordenador

es la acción que realiza una persona para manejar, estudiar,
trabajar, jugar o cualquier acción realizada ana maquina
tecnologica, por medio de un software y hardware.
